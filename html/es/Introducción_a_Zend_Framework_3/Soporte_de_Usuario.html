1.4. Soporte de Usuario

El soporte es un elemento importante a considerar cuando decidimos usar o no un framework como base para nuestro sitio web. El soporte incluye documentación bien escrita, conferencias web, foros comunitarios y (opcionalmente) servios de soporte comercial como cursos y programas de certificación.

Sitio web oficial de Zend Framework

Documentación. La documentación de Zend Framework está ubicada en esta dirección. Esta incluye tutoriales para aprendices y manuales de programación.

Referencia de la API se puede encontrar en este enlace.

Foro comunitario. Podemos hacer preguntas sobre como usar Zend Framework 3 en StackOverflow. Nuestras preguntas serán respondidas por una amplia comunidad de desarrolladores ZF3 como nosotros.

Conferencias webs. Son video tutoriales que cubren varias de las características de Zend Framework. Una lista completa de las conferencias se encuentra en este enlace.

Cursos de capacitación con instructores en vivo, se puede acceder a ellos por este enlace. Aquí podemos aprender Zend Framework haciendo ejercicios, pequeños proyectos y desarrollar código real.

Programas de Certificación. Nos permite convertirnos en un Ingeniero Zend Certificado (ZCE), con facilidad podemos mejorar nuestros habilidades como arquitecto y encontrar un trabajo dentro de un mercado competitivo de trabajo con PHP. Las certificaciones se encuentran aquí.
