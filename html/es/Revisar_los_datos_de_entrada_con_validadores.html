9. Revisar los datos de entrada con validadores

En este capítulo damos una visión general de los validadores estándares de ZF que se pueden usar en los formularios, además, mostraremos como escribir un validador a la medida. Un validador es una clase diseñada para tomar algunos datos, revisarlos para garantizar su corrección y regresar un booleano indicando si el dato es correcto y un mensaje de error si el dato tiene algunos errores.

Podemos usar validadores fuera de los formularios para procesar datos arbitrarios. Por ejemplo, los validadores se puede usar en las acciones del controlador para asegurar que los datos pasado en las variables GET y/o POST son seguras y cumplen con un determinado formato.

Los componentes de ZF3 tratados en este capítulo son:

Componente Descripción
Zend\Validator Implementa varias clases validadoras.
Zend\InputFilter Implementa un contenedor para filtros/validadores.
