Apéndice C. Introducción a Twitter Bootstrap

Twitter Bootstrap, o simplemente Bootstrap, es un popular framework CSS que permite que nuestro sitio web tenga un aspecto profesional y atractivo visualmente incluso si no tenemos habilidades avanzadas de diseñador.
