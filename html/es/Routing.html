5. Routing

Cuando un usuario del sitio escribe una URL en el navegador web, la petición HTTP se envía finalmente a una acción en un controlador de nuestro sitio web basado en ZF3. En este capítulo aprenderemos como una aplicación basada en ZF3 hace corresponder una URL a una acción de nuestro controlador. La correspondencia se logra con la ayuda del routing. El routing se implementa como una parte del componente Zend\Router.

ZF3 components covered in this chapter:

Componente Descripción
Zend\Router Implementa el soporte para routing.
Zend\Barcode Componente auxiliar que implementa código de barras.
