Apéndice B. Introducción al Desarrollo PHP con NetBeans IDE

En este libro usamos NetBeans IDE para desarrollar aplicaciones basadas en Zend Framework 3. En el Apéndice A. Configurar el Entorno de Desarrollo Web, hemos instalado NetBeans IDE. Ahora, daremos algunos consejos útiles cuando se usa NetBeans para la programación con PHP. Aprenderemos como usar la detección interactiva de errores en sitios web basados en ZF3.

¿Que pasa si queremos usar otro IDE y no NetBeans para desarrollar las aplicaciones?

Bien, podemos usar el IDE que queramos. El problema es que es imposible cubrir todos los IDEs para desarrollar PHP en este libro. Solo proveemos instrucciones para NetBeans IDE. Para un aprendiz puede ser fácil usar NetBeans IDE. Los desarrolladores avanzados pueden usar el IDE de su elección.
