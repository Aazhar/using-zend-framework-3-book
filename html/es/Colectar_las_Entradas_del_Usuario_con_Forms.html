7. Colectar las Entradas del Usuario con Forms

En este capítulo nos familiarizaremos con el uso de los formularios web para reunir los datos que el usuario ingresa al sitio web. En Zend Framework 3, la funcionalidad para trabajar con formularios está distribuida principalmente en cuatro componentes: el componente Zend\Form que permite construir formularios y contiene los ayudantes de vista para imprimir los elementos del formularios; los componentes Zend\Filter, Zend\Validator y Zend\InputFilter permiten filtrar y validar los datos ingresados por el usuario.

Componente Descripción
Zend\Form Contiene clases modelo para la base de los formularios.
Zend\Filter Contiene varias clases filtros.
Zend\Validator Implementa varias clases validadoras.
Zend\InputFilter Implementa un contenedor para filtros/validadores.
Zend\Mail Contiene la funcionalidad de envió de correo electrónico.
