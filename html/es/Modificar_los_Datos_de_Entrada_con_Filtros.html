8. Modificar los Datos de Entrada con Filtros

En este capítulo daremos una visión general de los filtros estándar que se pueden usar en nuestros formularios web. Un filtro es una clase que toma los datos de entrada, los procesa y produce algunos datos de salida. Además, mostraremos como escribir filtros personalizados.

En general, podemos usar filtros fuera de los formularios para procesar datos arbitrarios. Por ejemplo, los filtros se pueden usar en una acción del controlador para transformar los datos pasados por las variables GET y/o POST a un determinado formato.

Los componentes de ZF3 que se abarcan en este capítulo son:

Componente Descripción
Zend\Filter Contiene varias clases de filtros.
Zend\InputFilter Implementa un contenedor para filtros/validadores.
