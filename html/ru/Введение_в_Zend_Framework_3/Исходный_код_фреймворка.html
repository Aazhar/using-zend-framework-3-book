1.5. Исходный код фреймворка

Исходный код Zend Framework 3 хранится на хостинге GitHub внутри GIT-репозитариев. На каждый компонент ZF3 приходится по одному репозитарию.

В большинстве случаев вам не нужно скачивать код Zend Framework 3 вручную. Вместо этого, вы будете устанавливать его с помощью менеджера зависимостей Composer. Мы познакомимся с Composer поближе в главе под названием Zend Skeleton Application.

1.5.1. Стандарты оформления кода

Хорошей практикой является введение какого-либо стандарта оформления кода. Такой стандарт бы задавал правила для наименования классов, форматирования кода и т.д. В Zend Framework 3 есть такой стандарт - его можно прочитать здесь. Весь код в ZF3 следует правилам оформления, изложенным в том документе.

Если вы планируете писать веб-сайт на ZF3, рекомендуется использовать этот стандарт для вашего кода тоже. Это сделает ваш код "однородным" и другие люди смогут с легкостью его расширять и поддерживать.
