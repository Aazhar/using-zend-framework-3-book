Создание ключевой пары

Когда вы подключаетесь к виртуальной машине EC2, AWS использует ключевую пару для аутентификации. Эта пара состоит из открытого и закрытого ключей. Закрытый ключ нужно хранить в надежном месте и никогда и никому не давать.

Нажмите на икноку EC2 в окне AWS Management Console. Откроется страница EC2 Console ("Консоль EC2"). У нее есть боковое меню, содержащее такие разделы, как INSTANCES ("Инстансы), IMAGES ("Образы"), ELASTIC BLOCK STORE ("Эластичное блочное хранилище"), NETWORK & SECURITY ("Сеть и безопасность").

Чтобы безопасно подключиться к своему виртуальному серверу через протокол SSH, нужно создать ключевую пару. Для этого в боковом меню консоли EC2 в разделе NETWORK & SECURITY выберите пункт Key Pairs ("Ключевые пары").

Рисунок Д.3. Ключевые пары

На появившейся странице нажмите кнопку Create Key Pair ("Создать ключевую пару") и введите имя для создаваемой пары (например, test), затем нажмите кнопку Create ("Создать"). После того, как ключевая пара сгенерируется, вам будет предложено скачать ключ в виде PEM-файла. Сохраните этот файл в какой-нибудь каталог - позже он вам понадобится.
