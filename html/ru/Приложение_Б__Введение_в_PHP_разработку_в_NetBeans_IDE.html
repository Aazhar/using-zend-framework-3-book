Приложение Б. Введение в PHP-разработку в NetBeans IDE

В этой книге мы используем интегрированную среду разработки (Integrated development environment или IDE) NetBeans для разработки приложений на базе Zend Framework 3. В Приложении A. Настройка среды веб разработки мы установили среду NetBeans. В этом приложении мы дадим вам несколько полезных советов по использованию NetBeans для программирования на PHP. Так, мы научимся запуску и интерактивной отладке веб-сайта на базе ZF3.

Что если я хочу использовать другую IDE (не NetBeans) для разработки моих приложений?

Что ж, вы можете использовать любую среду разработки, которую хотите. Проблема в том, что рассмотреть все IDE для PHP-программирования в этой книге невозможно. Автор дает инструкции только для NetBeans. Начинающему будет проще использовать эту среду разработки. Продвинутые разработчики могут использовать IDE на свой вкус.

  • Конфигурация запуска
  • Запуск веб-сайта
  • Отладка сайта в NetBeans
  • Панель инструментов отладки
  • Точки останова
  • Наблюдение за переменными
  • Стек вызовов
  • Параметры отладки
  • Профилирование
  • Выводы
