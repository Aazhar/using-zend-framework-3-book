6. Apparence et mise en page

Dans ce chapitre, vous apprendrez comment rendre vos pages attrayantes et professionnelles avec l'aide du de la librairie CSS Twitter Bootstrap et comment positionner des éléments sur une page en utilisant le mécanisme de mise en page de ZF3. Vous vous familiariserez également avec les aides de vue communes permettant de composer des pages à partir d'extraits de code réutilisables. Si vous découvrez Twitter Bootstrap, il est également recommandé de se référer à l'[Annexe C. Introduction à Twitter Bootstrap](# bootstrap) pour une description avancée des fonctionnalités de Bootstrap.

ZF3 Composants traités dans ce chapitre :

Composants Description
Zend\Mvc Support of MVC pattern. Implements base controller classes, controller plugins, etc.
Zend\View Implements the functionality for variable containers, rendering a web page and common view helpers.
