Ce livre est destiné aux développeurs web impliqués dans le développement d'applications en PHP. L'auteur s'efforce de commencer avec des choses simples qu'un débutant devrait comprendre. Les choses compliquées arrivent à la fin dans les derniers chapitres. Ce qui en fait le premier livre sur Zend Framework accessible à un débutant. Vous n'avez pas besoin d'être un gourou en design patterns pour comprendre le contenu du livre.

Pour lire et comprendre ce livre, vous devez avoir quelques bases de PHP. Le mieux pour apprendre PHP est son site officiel et sa documentation en ligne L'idéal serait que vous ayez des notions sur les requêtes HTTP, les variables GET et POST, les namespaces, les classes et les interfaces.

Vu que PHP est étroitement lié à d'autres technologies web, il est également recommandé d'avoir les connaissances de base dans les domaines suivants:

  • HTML (Hyper Text Markup Language) -- utilisé pour créer des pages Web qui peuvent être affichées dans un navigateur Web.
  • CSS (Cascading Style Sheets) -- utilisé pour définir l'apparence d'une page Web, comme la taille de la police ou la couleur de fond.
  • JavaScript -- un langage de script utilisé pour rendre une page Web plus interactive.

Pour apprendre HTML, CSS and JavaScript, un bon point de départ sont les Tutoriaux W3Schools.
