5. Routage

Lorsqu'un utilisateur du site saisit une URL dans un navigateur, une requête HTTP est envoyée à l'action du contrôleur dans votre site basé sur ZF3. Dans ce chapitre, nous verrons comment l'application basée sur ZF3 mappe les URL aux contrôleurs et leurs actions. Cette cartographie (mappage) est réalisée à l'aide du routage. Le routage est implémenté avec le composant Zend\Router.

Les composants ZF3 traités dans ce chapitre :

Composant Description
Zend\Router Implémente le routage.
