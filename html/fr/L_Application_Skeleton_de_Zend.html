2. L'Application Skeleton de Zend

Zend Framework 3 vous fournit une application modèle plus connue sous l'appelation "application skeleton". Elle facilite la création de vos nouveaux sites web lorsque l'on part de zéro. Dans ce chapitre, nous allons voir comment l'installer et comment la faire tourner en créant un hôte virtuel Apache. Nous vous recommandons de vous reporter à l'Annexe A. Configuration d'un environnement de développement web avant de lire ce chapitre pour configurer votre environnement de développement.
