2. Zend Skeleton Application

O Zend Framework 3 fornece o que chamamos de "Skeleton Application" uma aplicação padrão que facilitar a criação dos seus novos sites a partir do zero. Neste capítulo, vamos mostrar como instalar o skeleton application e como criar um Apache virtual host.É recomendável antes de começarmos esse capítulo que seja feita a leitura do Appendix A. Configuring Web Development Environment onde ensina como deixar o ambiente desenvolvimento configurado e pronto para o desenvolvimento.
