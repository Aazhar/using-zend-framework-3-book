Sobre este Livro

O livro "Usando o Zend Framework 3" é um livro gratuito e de código aberto sobre a criação de sites com o Zend Framework 3. Este livro usa a licensa Creative Commons Attribution-NonCommercial-ShareAlike.

ZF3 é um framework de PHP moderno destinado a construir sites profissionais, escaláveis ​​e seguros. Esses sites são fáceis de testar e manter. O framework utiliza as melhores práticas e padrões comuns de design inspirados na evolução do desenvolvimento web. Isso inclui o padrão Model-View-Controller , que permite que o código de um aplicativo seja organizado de maneira consistente e padrão.

Este livro não é apenas sobre o Zend Framework, mas também sobre o estreitamente relacionado as suas bibliotecas. Embora o Zend Framework 3 tenha um componente dedicado para acessar o banco de dados, neste livro usamos uma biblioteca chamada Doctrine ORM que é um Maneira padrão orientada a objeto para executar o gerenciamento de banco de dados. Nos exemplos de aplicativos,que criaremos nos capítulos deste livro, O Framework CSS Twitter Bootstrap é usado para produzir páginas da Web bem desenhadas e organizadas.
