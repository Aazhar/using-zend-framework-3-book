Público Alvo

Este livro destina-se a desenvolvedores da Web envolvidos no desenvolvimento de aplicativos Web em PHP. O autor se esforça para começar com coisas simples que um iniciante deve Compreender. Isto faz deste livro o primeiro livro sobre o Zend Framework que é fácil de ler e entender para um novato. Você não precisa ser um guru em design patterns para entender a maior parte do conteúdo.

Para ler e entender este livro, você precisa ter um conhecimento básico de PHP. Um bom ponto para aprender PHP é seu site oficiale o on-line [documentação] (http://php.net/docs.php). È recomendavel que você entenda HTTP Requests, GET e POST, variáveis, namespace, classes, e interfaces.

Como o PHP está intimamente relacionado a outras tecnologias Web, também é recomendável Você tem alguma experiência básica no seguinte:

  • HTML (Hyper Text Markup Language) - usado para criar páginas da web que pode ser exibido em um navegador.
  • CSS (Cascading Style Sheets) - usado para definir a aparência de uma página, como o tamanho da fonte ou a cor do plano de fundo.
  • JavaScript - uma linguagem de script usada para fazer uma web página mais interativa.

Para aprender HTML, CSS e JavaScript, um bom ponto de partida é Tutoriais do W3Schools.
